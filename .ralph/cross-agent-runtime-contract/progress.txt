## Codebase Patterns
- Add only reusable patterns here.
- Shared agent policy should live in `.agents/rules/`.
- Shared hook logic should live in `.agents/hooks/`, but runtime adapter
  registration files remain runtime-specific.
- Repo-owned skills should stay in `.skills/` and be loaded on demand via
  `pnpm skills:list|search|read`.
