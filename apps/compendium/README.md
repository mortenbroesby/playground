# @playground/compendium

React + Next.js application that serves as a micro frontend orchestrator shell for apps in this monorepo.

## Scripts

- `pnpm --filter @playground/compendium dev` - run local development server on port 3001.
- `pnpm --filter @playground/compendium build` - produce a production build.
- `pnpm --filter @playground/compendium lint` - run Next.js lint checks.
- `pnpm --filter @playground/compendium type-check` - run TypeScript checks.
