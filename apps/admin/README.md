# @playground/admin

React + Vite admin surface for visualizing the repo task board from the vault.

## What it reads

- `vault/00 Repositories/playground/04 Tasks/Task Board.md`
- `vault/00 Repositories/playground/04 Tasks/tasks/*.md`

The app parses the board note plus the individual task notes and renders them as
a visual board for local maintenance work.

## Commands

- `pnpm --filter @playground/admin dev`
- `pnpm --filter @playground/admin build`
- `pnpm --filter @playground/admin preview`
- `pnpm --filter @playground/admin type-check`
- `pnpm --filter @playground/admin test`
