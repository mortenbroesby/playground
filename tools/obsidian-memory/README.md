# `@playground/obsidian-memory`

Repo-local Obsidian memory tooling for indexing the vault into `.rag/` and
retrieving architecture, decision, and session context.

## Commands

- `pnpm --filter @playground/obsidian-memory rag:init`
- `pnpm --filter @playground/obsidian-memory rag:index`
- `pnpm --filter @playground/obsidian-memory rag:query --query "<text>"`
- `pnpm --filter @playground/obsidian-memory rag:verify`
- `pnpm --filter @playground/obsidian-memory mcp`
- `pnpm --filter @playground/obsidian-memory test:retrieval`
