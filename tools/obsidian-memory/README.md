# `@playground/obsidian-memory`

Repo-local Obsidian memory tooling for building typed indexes under `.rag/`,
retrieving repo context, and maintaining note governance and cleanup flows.

## Commands

- `pnpm --filter @playground/obsidian-memory rag:init`
- `pnpm --filter @playground/obsidian-memory rag:index`
- `pnpm --filter @playground/obsidian-memory rag:classify`
- `pnpm --filter @playground/obsidian-memory rag:clean`
- `pnpm --filter @playground/obsidian-memory rag:doctor`
- `pnpm --filter @playground/obsidian-memory rag:fix-frontmatter`
- `pnpm --filter @playground/obsidian-memory rag:query --query "<text>"`
- `pnpm --filter @playground/obsidian-memory rag:test`
- `pnpm --filter @playground/obsidian-memory rag:verify`
- `pnpm --filter @playground/obsidian-memory rag:write`
- `pnpm --filter @playground/obsidian-memory mcp`
- `pnpm --filter @playground/obsidian-memory test:retrieval`
